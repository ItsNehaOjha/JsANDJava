Understanding DOM Selectors: NodeList & HTMLCollection

The Document Object Model (DOM) allows JavaScript to interact with HTML elements. Here, we explore different ways to select elements and manipulate them dynamically.

1️⃣ Selecting Elements by ID

Use document.getElementById("yourID") to get a single element by ID.

const title = document.getElementById("title");
title.style.color = "blue";

This is a Title

2️⃣ Selecting Elements by Class Name

Use document.getElementsByClassName("className") to get multiple elements.

const items = document.getElementsByClassName("item");
if (items.length > 0) {
    items[0].style.color = "red";
}

3️⃣ Query Selectors

Use document.querySelector("selector") to get the first match and querySelectorAll("selector") to get all matches.

const headings = document.querySelectorAll("h2");
headings.forEach(h => h.style.color = "#00A67E");

Effect: All h2 elements will now have their color changed to green (#00A67E).

4️⃣ Understanding NodeList vs HTMLCollection

  • NodeList: Returned by querySelectorAll, supports forEach but not array methods like map.
  • HTMLCollection: Returned by getElementsByClassName, does not support forEach.
  • Convert them to an array using Array.from() to use array methods.
const nodeListExample = document.querySelectorAll("h2");
nodeListExample.forEach(el => el.classList.add("highlight"));

const htmlCollectionExample = document.getElementsByClassName("item");
Array.from(htmlCollectionExample).forEach(el => el.style.fontWeight = "bold");
