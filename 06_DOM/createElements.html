Creating Elements with JavaScript

Use document.createElement("element") to create new elements.

Use element.appendChild(childElement) to add a child element to a parent element.

Use element.insertBefore(newElement, referenceElement) to insert a new element before a reference element.

Use element.replaceChild(newElement, oldElement) to replace an old element with a new element.

Use element.removeChild(childElement) to remove a child element from a parent element.

Use element.cloneNode(true) to clone an element and its children.

Use element.hasChildNodes() to check if an element has child nodes.

Use element.childNodes to get a NodeList of child nodes.

Use element.children to get an HTMLCollection of child elements.

Use element.firstElementChild to get the first child element.

Use element.lastElementChild to get the last child element.

Use element.previousElementSibling to get the previous sibling element.

Use element.nextElementSibling to get the next sibling element.

Use element.parentElement to get the parent element.

Use element.closest("selector") to get the closest ancestor element that matches the selector.

Use element.querySelector("selector") to get the first descendant element that matches the selector.

Use element.querySelectorAll("selector") to get all descendant elements that match the selector.

Use element.getAttribute("attribute") to get the value of an attribute.

Use element.setAttribute("attribute", "value") to set the value of an attribute.

Use element.hasAttribute("attribute") to check if an element has an attribute.

Use element.removeAttribute("attribute") to remove an attribute.

Use element.classList.add("class") to add a class to an element.

Use element.classList.remove("class") to remove a class from an element.

Use element.classList.toggle("class") to toggle a class on an element.

Use element.classList.contains("class") to check if an element has a class.

Use element.style.property = "value" to set a CSS property.

Use element.style.removeProperty("property") to remove a CSS property.

Use element.style.cssText = "property: value; property: value;" to set multiple CSS properties.

Use element.style.setProperty("property", "value") to set a CSS property with a priority.

Use element.style.getPropertyValue("property") to get the value of a CSS property.

Use element.style.getPropertyPriority("property") to get the priority of a CSS property.

Use element.style.removeProperty("property") to remove a CSS property.

Use element.style.length to get the number of CSS properties.

Use element.style.item(index) to get the property name at a specified index.

Use element.style.getPropertyValue("property") to get the value of a CSS property.

Use element.style.setProperty("property", "value") to set a CSS property with a priority.

Use element.style.getPropertyPriority("property") to get the priority of a CSS property.

Use element.style.removeProperty("property") to remove a CSS property.

Use element.style.length to get the number of CSS properties.

Use element.style.item(index) to get the property name at a specified index.
